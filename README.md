# Daily Commit Checker Bot
매일 6시 정각에 커밋 상태를 체크하고, 특정 사용자에게 상태 메세지를 전달합니다.

# Skills
Node.js, Typescript  
Discord.js v14

# Build
ESBuild

# Command
|Command|Description|
|---|---|
|Help|데일리 커밋 봇의 명령어를 가이드합니다.|
|Developer|데일리 커밋 봇 개발자를 소개합니다.|
|Search|검색한 사용자의 커밋 상태를 전달합니다.|
